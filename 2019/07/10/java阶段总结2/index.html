    内存分析

    栈空间（stack），连续的存储空间，遵循后进先出的原则，用于存放局部变量。
    堆空间（heap），不连续的空间，用于存放new出的对象，或者说是类的实例。
    方法区（method），方法区在堆空间内，用于存放①类的代码信息；②静态变量和方法；③常量池（字符串常量等，具有共享机制）。
