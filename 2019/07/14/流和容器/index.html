流和容器

流

  • 字节流

    • 字节输入流InputStream
      • 缓冲输入流BufferedInputStream
    • 字节输出流OutputStream
      • 缓冲输出流BufferedOutputStream

  • 字符流

    • 字符输入流Reader
    • 字符输出流Writer

  • 字节字符转换流

    • InputStreamReader
    • OutputStreamWriter

  • 其他功能流(节点流)

    • 数据流

    • 对象流

      • 对象输入流ObjectInputStream

      • 对象输出流ObjectOutputStream

        • 对象序列化

          • 方式(创建类,继承Serializable接口)
  • 测试字符拷贝
  • 测试对象流

容器

处理场景

  1. 存储数据数量不固定
  2. 同时处理有一对一关系(比如键值对)的数据
  3. 需要对数据进行处理
  4. 数据重复

List

特点

元素有序可以重复

使用

  • 增加add(index,element)
  • 删除remove(index)
  • 修改set(index,date)
  • 查询get(index) |indexof(Object)

遍历方式

  1. 循环(for,增强for)

  2. 迭代器(Iterator)

    迭代器(Iterator)

    Iterator接口用统一的方式对容器里元素进行遍历

    方法

    • hasNext()查看容器里是否还有下一个元素
    • next()返回容器中下一个元素
