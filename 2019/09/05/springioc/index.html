Spring 用处

dao:JDBC处理 Mybatis(CRUD) Hibernate

service:Spring 不是针对service 业务逻辑处理 没有适合的框架

controller:Servlet(表单参数接收 地址配置 页面转发) SpringMvc

Spring 基于分层的JavaEE 应用 轻量级JavaEE 框架(配置处理 Bean对象管理-IOC 集成 Mybatis|Hibernate SpringMvc Spring Security Quartz时钟框架 定时任务处理) 自带的服务(Mail 邮件处理 定时任务处理-定时调度(定时短信发送 定时邮件) 消息处理(异步处理))

Spring 框架模块划分

Spring IOC 模块:Bean 的实例化 Bean 的装配
Spring AOP 模块: 动态代理 面向切面编程
Spring JDBC+事物模块
Spring Web 模块
